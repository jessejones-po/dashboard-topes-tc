¿Cómo funciona?

  1. Ingresa el token de GitHub proporcionado por el equipo
  2. Selecciona el archivo CSV (ej. tc-davibank-ordenes-20260601-20260610.csv)
  3. Click en "Actualizar Dashboards"
  4. Espera ~3-5 minutos mientras GitHub regenera y publica los dashboards
  5. Recarga la página del dashboard para ver los datos nuevos
